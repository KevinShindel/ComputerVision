# Lesson 1

This folder contains the first lesson materials for an introduction to digital image processing in Python.

## Contents

- `homework/homework.md`  
  Homework instructions to set up the development environment (Anaconda/virtual environment) and install the core packages used in the course: `opencv`, `numpy`, and `matplotlib` (plus Jupyter Notebook).

- `Homework.ipynb`  
  A companion notebook for Homework 1 to verify the environment setup and practice running basic Python/Jupyter tasks.

- `ColorSpaces.ipynb`  
  Lesson notebook covering image loading and visualization, OpenCV’s BGR vs RGB channel order, and basic color space conversions (e.g., RGB ↔ LAB, grayscale).

## Requirements

- Python
- Packages: `opencv-python` (or `opencv` via conda), `numpy`, `matplotlib`, optionally `Pillow`
